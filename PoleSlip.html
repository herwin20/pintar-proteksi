Memahami Pole Slip Protection (ANSI 78) Memory Blinder Logic

Apa itu Pole Slip (Kehilangan Sinkronisasi)?

Pole Slip adalah kondisi ketika generator kehilangan sinkronisasi dengan sistem, di mana rotor berputar lebih cepat atau lebih lambat dari medan putar stator. Ini menyebabkan osilasi daya yang ekstrem dan dapat merusak generator. Proteksi ANSI 78 mendeteksi ini melalui trajectory impedansi yang membentuk "lensa" karakteristik.

👥 Analogi Tari Berpasangan

Generator dan sistem seperti pasangan penari. Pole slip adalah saat salah satu penari kehilangan irama dan mulai bergerak berlawanan arah. Proteksi seperti wasit yang mencatat setiap kali mereka saling membelakangi (blinder crossing).

🧒 Analogi Ayunan

Seperti anak di ayunan. Normalnya ayunan bergerak harmonis. Saat pole slip, ayunan berputar 360° melewati atas - ini berbahaya! Blinder seperti penanda batas aman ayunan.

🐉 Analogi Naga dan Cincin

Bayangkan naga mengejar ekornya sendiri. Saat sinkron, ia bergerak dalam lingkaran sempurna. Pole slip adalah saat ia mulai berputar liar. Blinder seperti cincin api yang tidak boleh dilewati dua kali.

Konsep MEMORY Blinder:

Logika yang benar: Begitu impedansi melewati blinder kiri atau kanan, kejadian itu DIINGAT (tidak hilang). Trip terjadi jika KEDUA blinder (kiri DAN kanan) pernah dilewati, menunjukkan satu siklus pole slip yang lengkap.

Rumus & Perhitungan

1. Impedansi Base

\(Z_{base} = \frac{V^2}{S}\)
V = 16 kV, S = 210 MVA 1.219 Ω

2. Impedansi Total ke Infinite Bus

\(X_{total} = (X_d' + X_{trafo} + X_{line}) \times Z_{base}\)
Xd' = 0.25 pu Xtrafo = 0.15 pu Xline = 0.40 pu
Xtotal = (0.25+0.15+0.40) × 1.219 0.975 Ω

3. Impedansi Saat Pole Slip

\(Z(\delta) = \frac{X_{total}}{2}(1 - e^{-j2\delta})\)
\(R = \frac{X_{total}}{2} \sin(2\delta)\)
\(X = \frac{X_{total}}{2} (1 - \cos(2\delta))\)

4. Setting Relay (Sekunder)

\(Z_{sek} = Z_{primer} \times \frac{CT_{rasio}}{VT_{rasio}}\)
CT = 2000:5, VT = 120:1 k = 2000/120 = 16.67
Xtotal sek = 0.975 × 16.67 16.25 Ω

5. Logika Trip (IEEE C37.102)

Trip jika: KEDUA blinder pernah terlewati
Memory L = pernah R < -3.5
Memory R = pernah R > 3.5

Kumpulan Rumus Lengkap Pole Slip Protection

1. Impedansi Base
\(Z_{base} = \frac{V_{LL}^2}{S_{3\phi}}\)
2. Total Impedansi
\(X_{total} = (X_d' + X_{trafo} + X_{line}) \times Z_{base}\)
3. Trajectory Pole Slip
\(Z(\delta) = j\frac{X_{total}}{2}(1 - e^{-j2\delta})\)
4. Komponen R dan X
\(R = \frac{X_{total}}{2}\sin(2\delta)\)
\(X = \frac{X_{total}}{2}(1-\cos(2\delta))\)
5. Konversi ke Sekunder
\(Z_{sek} = Z_{primer} \times \frac{CT_{rasio}}{VT_{rasio}}\)
